Hexo搭建个人博客

记性不好，博客还是要写起来，方便我寄几查看，2333

下载安装

  1. 下载node.js： 下载地址nodes.org
    查看安装版本：
    ‘’’shell
    node -v
    npm -v
    ‘’’
  2. 安装镜像源（加快速度）
    切换到管理员模式
    ‘’’shell
    sudo su
    ‘’’
    安装淘宝镜像源
    ‘’’shell
    npm install -g cnpm —registry=https://registry.npm.taobao.org
    ‘’’
  3. 安装hexo博客框架
    ‘’’shell
    cnpm install -g hexo-cli
    ‘’’
    查看版本
    ‘’’shell
    hexo -v
    ‘’’
    安装下载准备工作完成后就可以退出管理员模式了。

搭建博客

  1. 切换到你想要的路径建立文件夹
    ‘’’shell
    mkdir myblog
    ‘’’
  2. 进入blog目录
    ‘’’shell
    cd myblog/
    ‘’’
  3. 初始化一个博客
    ‘’’shell
    hexo init
    ‘’’
  4. 启动博客
    ‘’’shell
    hexo s // 先在本地看一下博客
    ‘’’
  5. 新建文章
    ‘’’shell
    // 新建
    hexo n “my_first_blog”
    // 进入新建文章所在的目录下
    ls
    cd source/_posts/
    ls -l
    // 修改添加内容
    vim my_first_blog.m
    // 保存退出 :wq
    ‘’’
  6. 退回到myblog目录下
    ‘’’shell
    cd ../..
    ‘’’
  7. 本地生成启动
    ‘’’shell
    // 清理
    hexo clean
    // 生成
    hexo g
    // 启动
    hexo s
    ‘’’

    部署到远端

  8. github建仓（注意仓库名字必须是:用户名.giyhub.io）
  9. 安装部署插件
    ‘’’shell
    cnpm install —save hexo-deployer-git
    ‘’’
  10. 设置config
    ‘’’
    vim _config.yml
    ‘’’
  11. 部署到远端
    ‘’’shell
    hexo d
    ‘’’

切换主题风格：

‘’’shell
vim _config.yml
‘’’
Theme:ilia
Hexo clean
Hexo g
Hexo s
Hexo d
